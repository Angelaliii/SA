@US-CMPM-REQ-001 @request-management @P0
功能: 需求發布
  身為平台上的社團/學生/企業/校友使用者
  我想要能夠發布我的合作需求或資源提供訊息
  以便尋找適合的合作對象或提供資源給需要的單位

  背景:
    假設我已成功登入社團企業媒合平台
    而且我處於個人儀表板頁面
    而且我點擊了"發布需求"按鈕

  @request-type-selection
  場景大綱: 需求類型選擇
    假設我的用戶角色是"<角色>"
    當我進入需求發布頁面
    那麼我應該看到適合我角色的需求類型選項
    當我選擇"<需求類型>"類型
    那麼系統應該顯示對應的需求表單
    當我不選擇需求類型就點擊"下一步"
    那麼系統應該顯示"請選擇需求類型"提示訊息

    例子:
      | 角色   | 需求類型     |
      | 社團   | 資源需求     |
      | 社團   | 人才需求     |
      | 社團   | 活動合作     |
      | 社團   | 技術支援     |
      | 學生   | 資源需求     |
      | 學生   | 專業諮詢     |
      | 學生   | 專題合作     |
      | 企業   | 資源提供     |
      | 企業   | 人才徵求     |
      | 企業   | 贊助合作     |
      | 校友   | 資源提供     |
      | 校友   | 經驗分享     |
      | 校友   | 導師計畫     |

  @basic-info-input

    而且我選擇期望完成日期"<完成日期>"

  @tag-and-category
  場景: 需求標籤與分類設定
    假設我已完成需求基本資訊填寫
    當我進入標籤與分類設定頁面
    那麼我應該看到預設標籤列表
    當我選擇標籤"設計"、"技術"和"行銷"
    而且我新增自訂標籤"社團合作"
    而且我選擇需求領域"教育"
    而且我設定需求優先級為"高"
    當我點擊"下一步"按鈕
    那麼系統應該成功進入下一步

    當我嘗試新增第6個標籤"多餘標籤"
    那麼系統應該顯示"最多可新增5個標籤"提示訊息

    當我清除所有領域選擇
    那麼系統應該顯示提示"建議選擇相關領域以提高媒合率"

  @attachment-upload
  場景: 附件與補充資料上傳
    假設我已完成需求標籤與分類設定
    當我進入附件與補充資料頁面
    那麼我應該看到檔案上傳區域
    
    當我上傳5MB的PDF企劃書"project_proposal.pdf"
    那麼系統應該顯示"檔案上傳成功"
    
    當我上傳15MB的影片檔案"large_video.mp4"
    那麼系統應該顯示"檔案大小超過限制，請壓縮後再上傳"
    
    當我上傳不支援的檔案格式"script.exe"
    那麼系統應該顯示"不支援此檔案格式"
    
    當我添加外部連結"https://example.com/resource"
    那麼系統應該接受此連結
    
    當我添加無效連結"invalid-link"
    那麼系統應該顯示"請輸入有效的網址"

  @publish-settings
  場景: 需求發布設定
    假設我已完成附件與補充資料上傳
    當我進入需求發布設定頁面
    
    那麼我應該看到公開範圍設定選項
    當我選擇公開範圍"僅對特定角色公開"
    而且我選擇可見角色"企業"和"校友"
    那麼系統應該接受此設定
    
    當我設定需求有效期限為45天
    那麼系統應該接受此設定
    
    當我設定需求有效期限為100天
    那麼系統應該顯示"建議設定不超過90天的有效期限"
    
    當我點擊"預覽需求"按鈕
    那麼系統應該顯示需求預覽頁面
    而且我應該看到所有已填寫的內容
    
    當我點擊"發布需求"按鈕
    那麼系統應該顯示"需求已成功發布"訊息
    而且系統應該導向我的需求列表頁面
    而且我應該在列表中看到新發布的需求

  @invitation-only
  場景: 設定邀請制需求
    假設我已完成附件與補充資料上傳
    當我進入需求發布設定頁面
    
    當我選擇公開範圍"邀請制"
    那麼我應該看到添加邀請對象的選項
    
    當我點擊"發布需求"按鈕而未添加任何邀請對象
    那麼系統應該顯示"請至少添加一個邀請對象"
    
    當我添加邀請對象"創新科技有限公司"
    而且我添加邀請對象"台大資管系學會"
    當我點擊"發布需求"按鈕
    那麼系統應該顯示"邀請制需求已成功發布"訊息
    而且系統應該向邀請對象發送通知
